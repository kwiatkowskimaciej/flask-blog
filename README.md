# Flask Blog
Backend for a bootstrap [blog template](https://startbootstrap.com/previews/clean-blog) based on Flask.

## About the project

### Features
- create an account
- login on account
- add new comment 
- add new post* 
- edit post*
- delete post* 

**Available only on Admin account*
